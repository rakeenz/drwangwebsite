Research Interests

  • Machine learning and data mining for effective decision making: information fusion, ensemble learning, and fuzzy control for detection and diagnosis; deep learning for image classification; and recurrent neural networks for sequential data predictive analysis.
  • Design, analysis, and optimization of complex engineering systems: discrete event simulation and agent-based modeling for complex system analysis; meta-heuristics, multi-objective evolutionary algorithms, and mixed integer programming for engineering process optimization.
  • Predictive dynamic control in smart manufacturing: smart electronics manufacturing using Internet of Things and machine learning; data-driven-based advanced manufacturing; fault operations and anomaly event diagnosis in high variation manufacturing processes.
  • Large-scale network system pattern recognition: community detection and network dynamics for practical large scale network analysis; temporal network for human brain functional connectivity pattern analysis.
